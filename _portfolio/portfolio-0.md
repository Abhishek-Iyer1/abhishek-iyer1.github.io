The different stages of this process are as follows:
- **Design Review**: Automatically detecting the focal image, and using an enhanced Median Color Quantization algorithm to pick key colors
- **Design Evaluation**: Using Monte Carlo Tree Search and Gaussian Mixture Models, we extend the palette with missing colors to create a harmony of complementing colors
- **Accessibility**: Assigning the colors from the palette to different layers of the UI design
- **Reports and Dashboard**: Checking certain metrics such as contrast, warm-cool combinations, luminance variations to ensure an appealing end result.
